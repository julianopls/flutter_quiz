## 📋 Sobre o Projeto

O **Flutter Quiz** é um aplicativo de perguntas e respostas construído com Flutter, projetado para ser executado em múltiplas plataformas com uma única base de código. Ideal para aprendizado de Flutter e para explorar os conceitos de gerenciamento de estado, navegação entre telas e widgets interativos.

---

## ✨ Funcionalidades

- 🎯 Perguntas de múltipla escolha
- 📊 Exibição de resultados ao final do quiz
- 🔄 Reinício do quiz a qualquer momento
- 📱 Interface responsiva e adaptável
- 🌐 Suporte a múltiplas plataformas

## 🚀 Como Executar

### Pré-requisitos

Certifique-se de ter o Flutter instalado na sua máquina. Caso ainda não tenha, siga o guia oficial: [flutter.dev/get-started](https://flutter.dev/get-started)

- **Flutter SDK:** `^3.11.0`
- **Dart SDK:** `^3.x`

### Instalação

```bash
# 1. Clone o repositório
git clone https://github.com/julianopls/flutter_quiz.git

# 2. Acesse a pasta do projeto
cd flutter_quiz

# 3. Instale as dependências
flutter pub get
```

### Executando o App

```bash
# Android / iOS
flutter run

# Web
flutter run -d chrome

# Windows
flutter run -d windows

# macOS
flutter run -d macos

# Linux
flutter run -d linux
```
